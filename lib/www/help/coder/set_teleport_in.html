The Third Age - Help set teleport in


 
   Type:        command	
   Usage:       set_teleport_in <text>
                
   Description: 'Set_teleport_in' is used to, as if you could not guess, set 
		your message when you teleport into a room!  You should note 
		that 'set_telelport_in' takes that place of your name in the 
		final output.  Don't forget the punctuation.

 
   Example:     > set_teleport_in steps out of a rift in space and time.
		(Others see when you teleport in)
		Erikose steps out of a rift in space and time.
   

 See also: 'set_walk_msg', 'set_run_msg', 'set_teleport_out', 'goto', 'review'
