The Third Age - Help set walk msg


 
   Type:        command	
   Usage:       set_walk_msg <text>
                
   Description: 'Set_walk_msg' is used to, as if you could not guess, set your 
		walk message!  You should note that 'set_rwalk_msg' takes the 
		place of your name and whatever you choose as your text should
		open ended to accept the name of the direction that you have 
		chosen to walk in.
		(set_walk_msg <message text> direction)
 
   Example:     > set_walk_msg waddles
		(Others see when you move)
		Erikose waddles east.
   

 See also: 'set_run_msg', 'set_teleport_in', 'set_teleport_out', 'review'
