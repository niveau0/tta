The Third Age - Help L


 
   Type:        command
   Usage:       L
                
   Description: The command 'L' is used by the Gods to give detailed 
		information about the objects in their vicinity.  'L'
		lists the information for objects, living and non-living,
		players/NPCs/objects in the same room/area as you.
 
   Example:     

  > L
  Name        Weight  Volume  Size Heap  Filename
  ==========================================================================
  manwe........94674...90860....85....0../config/race/ainu/ainu#48..........
  erikose......96000..105600....77....0../config/race/ainu/ainu#73..........
  cadderly....112000..123200....81....0../config/race/ainu/ainu#75..........
  endora......118000...94400....87....0../config/race/ainu/ainu#77..........
  bakerdoor....50000..100000....80....0../d/Standard/Bree/obj/baker_door#61.
  door.........55000..120000....90....0../d/Standard/Bree/obj/pub_door#63...
