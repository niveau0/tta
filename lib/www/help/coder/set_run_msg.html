The Third Age - Help set run msg


 
   Type:        command	
   Usage:       set_run_msg <text>
                
   Description: 'Set_run_msg' is used to, as if you could not guess, set your 
		run message!  You should note that 'set_run_msg' takes the 
		place of your name and whatever you choose as your text should
		open ended to accept the name of the direction that you have 
		chosen to run in.
		(set_run_msg <message text> direction)
 
   Example:     > set_run_msg zooms
		(Others see when you move)
		Erikose zooms north.
   

 See also: 'set_walk_msg', 'set_teleport_in', 'set_teleport_out', 'review'
