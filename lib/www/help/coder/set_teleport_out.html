The Third Age - Help set teleport out


   Type:        command	
   Usage:       set_teleport_out <text>
                
   Description: 'Set_teleport_out' is used to, as if you could not guess, set 
		your message when you teleport out of a room!  You should note 
		that 'set_telelport_out' takes that place of your name in the 
		final output.  Don't forget the punctuation.

 
   Example:     > set_teleport_out enters a rift and disappears.
		(Others see when you teleport out)
		Erikose enters a rift and disappears.
   

 See also: 'set_walk_msg', 'set_run_msg', 'set_teleport_in', 'goto', 'review'
