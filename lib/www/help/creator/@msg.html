The Third Age - Help @msg


   Type:        command
   Usage:       @msg [<channelname>@<ipname>] <message>
                
   Description: Send a message to a udp channel. If you want to choose a
		channel you need to specify channelname and ipname, further
		usage of @msg will send to the last chosen channel.

   Example:     @msg talk@tta Hello friends!
		@msg How are you?
   

 See also: '@status', '@open', '@join', '@list', '@close', '@ping', '@tell',
	   '@who'
