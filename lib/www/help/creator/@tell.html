The Third Age - Help @tell


   Type:        command
   Usage:       @tell <playername>@<ipname> <message>
                
   Description: Send a message to someone at a specified ipname. 

   Example:     @tell manwe@tta Hello!
   

 See also: '@status', '@open', '@close', '@join', '@msg', '@list', '@ping',
	   '@who'
