The Third Age - Help @join


   Type:        command
   Usage:       @join <channelname>@<ipname>
                
   Description: Join an udp channel. You need to specify the channelname 
		one of the ipnames, which can be found in the '@status' 
		list.

   Example:     @join generaltalk@tta
   

 See also: '@status', '@open', '@close', '@list', '@msg', '@ping', '@tell',
	   '@who'
